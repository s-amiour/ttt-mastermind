# Sprint 1

## Sprint Goal
Deliver a working MVP of the Tic-Tac-Toe game with the core interface and game logic.

## Focus
- Structure and Design
- Game Engine Functionality

## Planned Work
- Render the 3x3 board
- Implement player interaction
- Prevent moves on occupied cells
- Detect win conditions
- Detect draw
- Display game result
- Restart game
- Implement initial AI logic

## Definition of Done
- Feature is implemented
- Feature is tested
- Code is pushed to repository
- Demo is ready for Sprint Review
